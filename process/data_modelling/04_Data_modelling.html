Per tal de trobar la similitud que hi ha entre els diferents estudis clnics, s'executa l'algoritme del coeficient de Jaccard. Aquest algoritme de la llibreria Networkx executa Jaccard entre dos parells de nodes, en aquest cas, per a cada parell d'estudis clnics, i on el resultat de similitud depn del nmero de propietats que tenen iguals i el nmero total de propietats entre els dos.

Per exemple, en la imatge segent es mostra que el resultat del coeficient Jaccard s de 0.84, on comparteixen 11 propietats de 13.

S'executa el coeficient per a cada parell d'estudis clnics que hi ha en el dataset. Tot i aix, noms es guarden els que tenen una similitud ms gran o igual a 0.5.

THRESHOLD = 0.5


def get_jaccard_coefidient(graph):
    """ Mtode que ejecuta el Jaccard coeficient a un graf i retorna les similituds ms 
    gran que el threshold i les semblanes entre els estudis clnics.

    Parameters:
        graph: Graf de Networkx

    Returns:
        List: parells d'estudis clnics i el grau de similitud, de forma estructurada.
    """
    result_text = []
    result_structured = []

    results = nx.jaccard_coefficient(graph)
    
    for u, v, p in results:
        # Noms es retornen les relacions entre els estudis clnics.
        if u.startswith("ClinicalTrial::") and v.startswith("ClinicalTrial::"):
            # Per damunt el threshold
            if p >= THRESHOLD:
                result_structured.append([u, v, p])
                
    return result_structured

Cerca de comunitats¶

Una comunitat est formada per un conjunt d'estudis clnics que sn semblants pel que fa a les segents caracterstiques:

  • Objectiu de l'estudi: tractament o observacional.
  • Tipus d'intervenci.
  • Condici de salut que s'est estudiant.
  • Les paraules clau extretes de forma automtica i que utilitzen la taxonomia Mesh.


Tots els estudis clnics d'una mateixa comunitat han de tenir una similitud igual o superior a 0.6.

PATH_DESTINATION_COMMUNITIES = '../../results/communities/'

clinical_trial_processed = []
list_communities = {}
community_id = 0

THRESHOLD = 0.6

def get_community(current_clinical_study, current_community):
    """ Mtode recursiu per a obtenir la comunitat a partir d'un estudi clnic.

    Parameters:
        current_clinical_study (str): Identificador de l'estudi clnic concret.
        current_community (list): Llistat amb els estudis clnics que pertanyen a una comunitat.

    Returns:
        List: Llistat amb els estudis clnics que pertanyen a una comunitat.
    """
    
    # Estudis clnics similars al current_clinical_study, que no s'hagin processat ni descartat per aquesta comunitat.
    communities_a = [x[1] for x in clinical_trials_similarity 
                     if x[0] == current_clinical_study and x[2] >= THRESHOLD 
                     and x[1] not in clinical_trial_processed 
                     and x[1] not in clinical_trials_discarded_community 
                     and x[1] not in current_community]
    
    communities_b = [x[0] for x in clinical_trials_similarity 
                     if x[1] == current_clinical_study and x[2] >= THRESHOLD 
                     and x[0] not in clinical_trial_processed 
                     and x[0] not in clinical_trials_discarded_community 
                     and x[0] not in current_community]
    
    communities = communities_a + communities_b if len(communities_b)>1 else communities_a
    
    for community in communities:
        if community not in clinical_trials_discarded_community and community not in current_community:
            continue_process = True

            # Validem que l'estudi clnic similar, tamb sigui similar amb els altres estudis clnics dins de la comunitat
            for community_processed in current_community:
                if community_processed != current_clinical_study and community_processed not in clinical_trials_discarded_community:
                    score = [x[2] for x in clinical_trials_similarity 
                             if (x[0] == community and x[1] == community_processed)
                             or (x[1] == community and x[0] == community_processed)]

                    score = score[0] if len(score)>0 else 0

                    if score < THRESHOLD:
                        continue_process = False

            if continue_process:
                current_community.append(community)
                get_community(community, current_community)
            else:
                clinical_trials_discarded_community.append(community)
            
    return current_community
                    

for triple in clinical_trials_similarity:
    # Extrec l'identificador de l'estudi clnic.
    clinical_trial = triple[0]
    
    clinical_trials_discarded_community = []
    # Si l'estudi clnic encara no ha estat processat, continuo.
    if clinical_trial not in clinical_trial_processed:
        # Genero la comunitat, amb l'estudi clnic com el primer de la comunitat.
        list_community = [clinical_trial]
        # Obtinc altres estudis clnics similars per a la comunitat.
        list_community = get_community(clinical_trial, list_community)
        
        # Si la funci get_community retorna un altre estudi clnic, es genera la comunitat.
        # Sin es descarta.
        clinical_trial_processed = clinical_trial_processed + list_community if len(list_community)>0 else clinical_trial_processed

        # Guardo els estudis clnics de la comunitat en un fitxer.
        if len(list_community) > 1:
            filename = PATH_DESTINATION_COMMUNITIES + str(community_id) + '.txt'
            with open(filename, 'w', encoding='utf-8') as f:
                for r in list_community:
                    f.write(r) 
                    f.write('\n')

            community_id += 1
            list_community = []
            clinical_trials_discarded_community = []
